# i18n ルール

## ファイル

- `src/lib/i18n/ja.json` — 日本語（デフォルト）
- `src/lib/i18n/en.json` — 英語

## 鉄則

**新しい翻訳キーを追加するときは必ず ja.json と en.json の両方に追加する。**
片方だけ追加すると、もう片方の言語で表示が崩れる。

## 言語判定は必ず `$locale` ストアを使う

```typescript
// NG: 非リアクティブ。言語切替後に再評価されない
const isEn = document.documentElement.lang === 'en';

// OK: $locale はリアクティブストアなので Svelte テンプレートで自動追従
import { locale } from '$lib/i18n';
// テンプレート内: {$locale === 'en' ? '...' : '...'}
```

## コーデックラベルの翻訳パターン

`src/lib/ffmpeg/codecs.ts` の `VIDEO_CODEC_LABELS` / `AUDIO_CODEC_LABELS` は
技術名（H.264, AAC 等）以外の `copy` / `pcm_s16le` が日本語ハードコード。
コンポーネント側で **`$derived` を使って翻訳済みラベルマップを生成し上書き** する。

```typescript
// PresetCustomizer.svelte パターン
const videoCodecLabels = $derived({
  ...VIDEO_CODEC_LABELS,
  copy: $t('form.codecCopy')
});
const audioCodecLabels = $derived({
  ...AUDIO_CODEC_LABELS,
  copy: $t('form.codecCopy'),
  pcm_s16le: $t('form.codecPcmUncompressed')
});
```

## Storybook での i18n

`.storybook/preview.ts` のデコレータが `locale.set(loc)` を呼び出すため、
ストーリー内で `$t()` は通常通り使える。ツールバーの Globe アイコンで ja/en 切替が可能。
ストーリー側で追加の i18n セットアップは不要。
