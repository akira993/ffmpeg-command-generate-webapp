# i18n ルール

## ファイル

- `src/lib/i18n/ja.json` — 日本語（デフォルト）
- `src/lib/i18n/en.json` — 英語

## 鉄則

**新しい翻訳キーを追加するときは必ず ja.json と en.json の両方に追加する。**
片方だけ追加すると、もう片方の言語で表示が崩れる。

## キー構造の概要

```
common.{copy, copied, reset, input, output, apply}
header.{title, subtitle}
mode.{preset, advanced}
preset.{videoConvert, videoCompress, audioExtract, audioConvert, videoTrim, gifGenerate, imageConvert}.{name, desc}
form.{filename, format, codec, resolution, framerate, bitrate, crf, sampleRate, channels, startTime, endTime, duration, noVideo, noAudio, stripMetadata, copyStreams}
command.{generated, empty, copied}
pathGuide.{buttonLabel, title, description, step1〜4, tipTitle, tip{Mac,Windows,Linux}, warning, installPrompt, installLink}
installGuide.{buttonLabel, title, description, mac.*, windows.*, verifyTitle, verifyDesc, officialSite}
```

## 補間構文

sveltekit-i18n の補間は `{{variable}}` 形式。
